==============================
Writenc - NetCDF File Creation
==============================

The `writenc` module provides functionality for creating and manipulating NetCDF files from ADCP binary data. It includes methods for adding global attributes, creating NetCDF dimensions and variables, and writing processed data to NetCDF files.
