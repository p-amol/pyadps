**ADCP Data Processing Software (ADPS)**
========================================
ADPS is a software for processing RDI Acoustic Doppler Current Profiler (ADCP) PD0 files. Currently the software can process the data from Workhorse and DVS. We are working on including navigation file and multiple file processing.
